Cuales son los pasos para realizar un anlisis exploratorio de datos?

Los podemos dividir principalmente en 5 pasos:

  1. Hacer preguntas.
  2. Determinar el tamao de los datos.
  3. Categorizar las variables.
  4. Limpieza y validacin de datos.
  5. Establecer relaciones.

Hacer preguntas

Necesitamos tener preguntas para nuestros datos. Una vez visualicemos el conjunto de datos, empecemos a hacernos preguntas, es decir toda clase de preguntas:

  • qu te gustara encontrar en tus datos?
  • qu quieras saber sobre los datos?
  • cul es la razn para realizar el anlisis?

A traves de generar estas preguntas seremos capaces de dar una gua a todo el proceso de exploracin de datos.

Determinar el tamao de los datos

Con este paso nos vamos a hacer preguntas como:

  • cuantas observaciones existen?
  • cuantas variables hay?
  • necesito todas las observaciones?
  • necesito todas las variables? Digamos podr tener variables que no me aportan informacin til?, este tipo de preguntas sern muy recurrentes.

Categorizar nuestras variables

Aqu nos vamos a preguntar:

  • cuntas variables categricas existen?
  • cuntas variables continuas existen?
  • cmo puedo explorar cada variable dependiendo de su categora? Es decir preguntarnos y cuestionarnos que anlisis puedo hacer.

Limpieza y validacin de datos

Ahora que ya sabemos cual es el tamao de los datos, cuantas variables hay. Nos tenemos que preguntar:

  • tengo valores faltantes?
  • cul es la proporcin de datos faltantes?
  • cmo puedo tratar los datos faltantes?

  • tengo valores atpicos? Es decir ver si hay una razn por la que mis datos estn ausentes, y en caso de que falten ser que puedo rellenarlos con otros valores de mi conjunto?, as podemos cuestionarnos para saber como lidiar con ellos. Tambin es bueno para saber que hacer con valores que estn en los extremos, ya sea muy altos o muy pequeos.

Este paso nos va a ayudar a determinar la respuesta a todas estas preguntas.

Establecer relaciones

Finalmente tenemos un paso en el que vamos a establecer las relaciones entre nuestros conjuntos de datos y nos vamos a hacer preguntas como:

  • existe un relacin entre mi variable X e Y?
  • qu pasa si ahora considero a la variable Z en mi anlisis?
  • qu significa que las observaciones se agrupen?
  • qu significa el patrn que se observa? Cosas extraas podran pasar en este proceso, tambin podremos preguntarnos que pasa con las proporciones; es decir para un conjunto de datos tengo miles de observaciones y para otra solo tengo unas cuantas. Es decir buscar si hay o no explicaciones con mis datos a los patrones que presenten, es decir para encontrar o extraer ms informacin perdida dentro de mis datos y as dar una respuesta a estas preguntas.

Estos son los pasos que debemos seguir, pero hay que contemplar que no es un proceso lineal, mas bien puede parecer un proceso cclico. Y podremos repetir ciertos pasos nuevamente, es decir:

Una vez que terminemos lo vamos a volver a realizar; vamos a filtrar datos, realizar nuevas preguntas, volver a calcular el tamao de los datos y as nuevamente.

A pesar de que pueda parecer un ciclo infinito y que no se acaba seguir en el no tiene sentido, y tiene que existir un momento en el que debemos romper el ciclo y continuar.

Un anlisis exploratorio de datos que nunca termina en realidad tiene un valor nulo; porque nunca le pudimos entregar la informacin a alguien o nunca pudimos realizar algo que genere impacto con estos datos en la empresa, en el trabajo o en alguna determinada razn que tengamos para realizarlo.
