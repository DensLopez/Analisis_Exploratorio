Conociendo nuestros datos: palmerpenguins

Estos datos son sobre pinginos y provienen de The Palmer Long-Term Ecological Research (LTER), que se dedican a colectar datos sobre la flora y fauna en la Antrtida. Un caso especifico son los pinginos.

Este set de datos tiene una gran aplicacin para hacer anlisis de datos, adems que ayuda a entender ciertos parmetros que necesitamos para el anlisis exploratorio de datos.

Especies

En particular se dedicaron a tomar muestras de 3 especies:

  1. Chinstrapi
  2. Gentoo
  3. Adlie

Algo particular de los pinginos diferente a otros animales, es la determinacin del sexo; pues requiere de anlisis de sangre y otros parmetros, para poder determinarlo. As los investigadores se pusieron a indagar para poder ayudar a determinar.

Longitud del pico [Bill length] y altura de pico [bill depth] son parmetros que se midieron, ademas de otros parmetros. As apoyndose de estos datos y de pruebas genticas relacionaron los datos con su determinado sexo, para despus con el anlisis de datos hacer una correlacin y de esta manera poder predecir mediante las medidas de ciertos parmetros; si un pingino es macho o hembra.

Recuerda que un anlisis de datos se basa en hacer preguntas, as que trata de preguntarte cosas sobre lo que vas a trabajar; en este caso pinginos. Podremos preguntarnos, como fue su proceso evolutivo? Cuntas especies distintas hay? Qu diferencia a cada especie? etc.

De esta manera es bueno mostrar inters en los datos que vamos a analizar para que sea un proceso mas natural, as podemos realizar las siguientes preguntas:

  • sera posible determinar el sexo de un pingino por las medidas de su pico?
  • qu tanto puede vivir un pingino?
  • qu tanto puede pesar un pingino?

Es bueno tener este tipo de preguntas, dado que conforme pase el proceso de anlisis, podemos regresar y dar respuesta a ese tipo de preguntas.
